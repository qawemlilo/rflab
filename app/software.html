Tools, Software & Apps

CrushIt

CrushIt is a command-line software for compiling, concatenating, and minifying scripts from web pages

CrushIt Compiler

CrushIt Compiler is a web interface for CrushIt.

Node-Ping

Node-ping is an Open Source website monitoring tool, it checks the availabilty of a website and sends out an email if it is down.

NodeTube

NodeTube is a web app for downloading YouTube videos. I created NodeTube because I needed to download tutorial videos for an online course I was doing.
